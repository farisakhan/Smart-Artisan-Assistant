P1. Smart Artisan Assistant (Simple Web App + AI)
Problem Statement
Artisans currently manage their work (production, materials, payments) using notebooks or WhatsApp, making it hard to track and analyze.
Build a simple web application where artisans can:
Add daily production entries (items made, materials used)
Track their earnings and payment history
View simple reports (weekly/monthly summary)

AI Feature (Simple but useful)
User uploads a product image → system suggests:
Estimated price range
Possible defects (basic image-based hints)
OR AI chatbot to answer: “How can I improve my product quality?”

Goal
Make daily tracking easy and give basic AI-driven guidance.
